Models Validation
=================

How to Install the python library
---------------------------------
::

    cd /path_to/GelReportModels
    (sudo) pip install . (--upgrade)

How to use the Validation script
--------------------------------
::

    ModelValidatior <json_file> <schema_model> <version_model>

.. note::
    If you want to use the last version version_model parameter should be set as **`latest`**
